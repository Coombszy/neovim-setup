# Neovim config

## Installation (Ubuntu)
1. Install custom font in `fonts` folder (And set terminal to use it)
2. Install packages:
```sh
sudo apt update
sudo apt install fzf ripgrep
```
3. Copy init.lua into config folder:
```sh
cp -R init.lua ~/.config/nvim/init.lua
```
